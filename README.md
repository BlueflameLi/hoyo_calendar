# 🗓️ 米哈游游戏日历订阅

![Banner Image](./docs/public/banner.png)

✨ 自动同步《原神》《星穹铁道》《绝区零》的官方活动日程到您的日历应用

## 📥 订阅方法

1. **复制**下方需要的日历链接
2. **打开**您使用的日历应用（支持Google日历、Outlook、苹果日历等）
3. 找到**订阅日历**功能，粘贴URL完成订阅
